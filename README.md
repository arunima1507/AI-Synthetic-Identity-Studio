# AI Synthetic Identity Studio

## Problem Statement

Real-world human datasets are difficult to share because of privacy concerns and legal restrictions.

This project aims to generate synthetic human identities consisting of AI-generated face images and demographic profiles that can be used for:

- Software testing
- UI/UX prototyping
- Machine learning experimentation
- Educational purposes

without exposing real personal information.
